jlab_jaws.clients.JAWSProducer¶

class jlab_jaws.clients.JAWSProducer(topic, client_name, key_serde, value_serde)[source]¶

Bases: object

This class produces messages to JAWS.

The JAWS expected header is included in all messages.

Sensible defaults are used to determine BOOTSTRAP_SERVERS (look in env) and to handle errors (log them).

This producer also knows how to import records from a file using the JAWS expected file format.
