Instance is a dict with one entry of key type name str and value dict; this serializes to JSON as AVRO expects (except for logical types and bytes) - See: http://avro.apache.org/docs/current/spec.html#json_encoding

POSSIBLY_AMBIGUOUS_DICT

Instance dict provided without type name - which in the case of records with identical fields there is no way to determine which is which (example: union of classes A and B where each has identical fields)
